# Agency.ai

Modern AI agency website built with React, Vite and Tailwind CSS.

## Live Demo

https://agency-ai-cyan-nu.vercel.app

## Features

- Fully responsive design
- Dark mode support
- Animated UI with Framer Motion
- AI service planner section
- Interactive project showcase modal
- Functional contact form
- Premium startup-style interface
- Mobile optimized experience

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Web3Forms
- React Hot Toast
